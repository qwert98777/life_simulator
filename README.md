
# Life Simulator

Консольный симулятор жизни на Python. Проект для курса "Код будущего".

## Описание

Симуляция экосистемы, где хищники охотятся на травоядных, все организмы питаются, тратят энергию и размножаются.

## Структура проекта
life_simulator/
├── main.py       # точка входа
├── ecosystem.py  # основная логика симуляции
├── organism.py   # классы организмов
├── .gitignore    # исключения для Git
└── README.md     # описание проекта
